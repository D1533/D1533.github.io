## Pong in x86 ASM
---

The main goal of this project was to learn assembly, so more experienced assembly programmers might find many non-optimized functions or project structures. Usually, it doesn’t make sense to reinvent the wheel, but for purely educational projects, it can be very valuable. That's why I avoided using any C libraries throughout the project. This means that functions like print(), len() and others had to be implemented from scratch. A basic AI, if you can call it that, is also included, which simply follows the ball once certain conditions are met.
